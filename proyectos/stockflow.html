Proyecto Principal

StockFlow API REST

Sistema de gestión de inventarios construido con Spring Boot y PostgreSQL, funcionando además como punto de venta (POS) con registro de ventas e historial de movimientos.

Ver código Ver demo

Resumen

StockFlow API REST es un sistema de gestión de inventarios pensado para pequeños y medianos negocios: administra proveedores, catálogo de productos y niveles de stock desde un único backend.

Además de la gestión de inventario, StockFlow funciona como punto de venta (POS): registra ventas, actualiza el stock automáticamente y mantiene un historial completo de movimientos para trazabilidad total.

Funcionalidades clave

Gestión de Proveedores

Alta, edición y consulta de proveedores asociados a los productos del inventario.

Catálogo de Productos y Stock

Control de existencias en tiempo real, con actualización automática ante cada movimiento.

Punto de Venta (POS)

Registro de ventas que descuenta stock automáticamente y calcula totales al momento.

Historial de Movimientos

Trazabilidad completa de entradas y salidas de inventario para auditoría y reportes.

Stack Tecnológico

Java Spring Boot PostgreSQL Docker JPA Hibernate Spring Web Swagger / OpenAPI

Arquitectura

  • API REST construida con Spring Boot y Spring Web
  • Persistencia con JPA / Hibernate sobre PostgreSQL
  • Documentación interactiva con Swagger / OpenAPI
  • Contenedorizada con Docker y desplegada en el home lab personal
